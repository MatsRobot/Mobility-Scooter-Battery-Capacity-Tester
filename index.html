A specialized diagnostic tool designed to measure the real-world health of Lead-Acid (AGM) batteries. It uses a simulated 5A load and automated Arduino logging to determine the true discharge curve without damaging the cells.

Critical Threshold: Lead-acid batteries suffer irreversible damage if discharged below 30%. This system automatically disconnects the load at the 50% "Safe Limit" (12.23V) to preserve battery longevity while measuring capacity.

How It Works: Load Simulation & Logic

To accurately simulate a mobility scooter in motion, the device uses a 110W resistive load (dual 12V-55W halogen bulbs). The Arduino performs the following low-level logic:

  • No-Load Measurements: The system temporarily switches off the load during every measurement cycle. This is vital because battery state-of-charge (SoC) voltages are only accurate when the battery is "resting" without a voltage drop.
  • Series Subtraction Math: Since mobility scooters use 24V (two 12V batteries in series), the code measures the total voltage ($V_{total}$) and the first battery ($V_1$). It then calculates the second battery via $V_2 = V_{total} - V_1$.
  • Thermal Management: Due to the high heat output of the resistive load, bulbs are isolated in glass housings with dedicated fan cooling.

Rugged Power Handling

Designed to handle the 21A peak currents measured during real-world grass riding. The 5A test load provides a highly accurate average-use simulation.

Accurate Health Check

While standard CCA (Cold Cranking Amp) testers give a "snapshot" of health, this device measures actual Amp-Hour capacity over 5-6 hours of continuous testing.

Voltage-to-Capacity Reference

The system is programmed with the following reference values for 12V Lead-Acid batteries to ensure precise logging:

  • 100% (Full): 12.89V
  • 50% (Safe Limit): 12.23V
  • 30% (Damage Risk): 11.96V
  • 0% (Dead): 11.63V
